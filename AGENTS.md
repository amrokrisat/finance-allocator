# AGENTS.md

Project goal:
Build a personal finance allocation tool that tells the user exactly how to allocate income each month.

Rules:
- Prioritize correct financial logic over UI polish.
- Keep calculations deterministic where possible.
- Never recommend aggressive investing when high-interest debt and low liquidity exist.
- Build modular services.
- Add tests for all core allocation logic.
- Explain assumptions in comments and the README.
- Keep the MVP simple.

Output expectations:
- Monthly allocation plan
- Per-paycheck allocation plan
- Debt payoff summary
- Savings and investing summary
- Risk assessment
- Clear next-step roadmap
